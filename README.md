# StubbornKnight - 空洞骑士 Mod

[**English Version**](README-en.md) | **中文说明**

## 🎮 游戏机制

### 核心玩法
StubbornKnight 是一个**偏执骑士 Mod**。小骑士的内心始终执着于一个方向，你只能朝它想的方向发起攻击。

### 箭头系统
- 玩家头顶显示 **4 种垂直排列的箭头**（上/下/左/右）
- **最底部的箭头**是当前小骑士内心想的方向
- 成功执行动作后，箭头队列会向下滚动，生成新的方向

### 动作规则

| 动作类型 | 操作方式 | 要求方向 |
|---------|---------|---------|
| **普通攻击** | 按攻击键 | 左/右（根据角色朝向） |
| **蓄力斩** | 攻击 + 蓄力 | 左/右（根据角色朝向） |
| **冲刺劈砍** | 攻击 + 冲刺 + 蓄力 | 左/右（根据角色朝向） |
| **旋风斩** | 攻击 + 上/下 + 蓄力 | 上/下 |
| **波** | 法术键 | 左/右（根据角色朝向） |
| **吼** | 法术键 + 上 | 上 ⬆️ |
| **砸** | 法术键 + 下 | 下 ⬇️ |

### 失败惩罚
- 动作方向与内心方向**不符**时，动作将被**拦截**（不产生攻击判定/不消耗法力）
- 箭头不会滚动，你需要调整方向重新尝试

### 成功
- 方向匹配后，动作正常释放
- 箭头队列滚动，进入下一个方向

## ⚙️ 设置

在游戏主菜单的 Mod 选项中可以调整以下设置：

| 设置项 | 说明 | 范围 |
|--------|------|------|
| **StubbornKnight** | Mod 总开关 | 开启 / 关闭 |
| **箭头队列长度 (Arrow Queue Length)** | 玩家头顶显示的箭头数量 | 1 - 30 |
| **箭头不透明度 (Arrow Opacity)** | 箭头显示的透明度 | 0.1 - 1.0 |
| **音效音量 (Sound Volume)** | 方向匹配成功时的提示音效音量 | 0 - 10 |

Mod 关闭时，所有动作不受方向限制，恢复原版游戏行为。
