# StubbornKnight - 空洞骑士节奏 Mod

## 🎮 游戏机制

### 核心玩法
StubbornKnight 是一个**节奏动作 Mod**，为空洞骑士添加了方向匹配机制。玩家需要在执行动作时匹配当前显示的方向箭头，否则动作将被**拦截**。

### 箭头系统
- 玩家头顶显示 **4 个垂直排列的箭头**（上/下/左/右）
- **最底部的箭头**是当前需要匹配的目标方向（半透明）
- 成功执行动作后，箭头队列会向上滚动，生成新的目标方向

### 动作匹配规则

| 动作类型 | 操作方式 | 匹配方向 |
|---------|---------|---------|
| **普通攻击** | 按攻击键 | 左/右（根据角色朝向） |
| **蓄力斩** | 攻击 + 蓄力 | 左/右（根据角色朝向） |
| **冲刺劈砍** | 攻击 + 下 + 蓄力 | 下 ⬇️ |
| **旋风斩** | 攻击 + 上 + 蓄力 | 上 ⬆️ |
| **复仇之魂/暗影之魂** | 法术键 | 左/右（根据角色朝向） |
| **嚎叫幽灵** | 法术键 + 上 | 上 ⬆️ |
| **深渊尖啸** | 法术键 + 下 | 下 ⬇️ |

### 失败惩罚
- 动作方向与目标箭头**不匹配**时，动作将被**完全拦截**（不产生攻击判定/不消耗法力）
- 箭头不会滚动，玩家需要调整方向重新尝试

### 胜利奖励
- 成功匹配方向后，动作正常释放
- 箭头队列滚动，进入下一个节奏循环

## ⚙️ 设置
- 在游戏主菜单的 Mod 选项中可以**开启/关闭**此 Mod
- Mod 关闭时，所有动作不受方向限制，恢复原版游戏行为

## 🎯 技巧提示
1. 时刻关注底部箭头方向
2. 普通攻击和蓄力斩的方向由角色**面朝方向**决定
3. 法术和特殊剑技需要**提前输入对应方向**
4. 可以在神居或任意安全区域练习节奏感

## 📋 版本
- 当前版本：1.0
- 支持游戏版本：1.5.x（空洞骑士）

## 🔧 安装
确保已安装 **Scarab** 或 **xUnity Auto Translator** Mod 加载器，将 Mod 文件放入 `hollow_knight_Data/Managed/Mods/StubbornKnight/` 目录即可。
