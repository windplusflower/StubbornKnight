## 2. Architecture (INNOVATE)
- **Pattern**: FSM 注入 + Action 回调
- **Strategy**: 
  - 在 `PlayMakerFSM_OnEnable` 中检测 `FsmName == "Nail Arts"`
  - 使用 `InsertMethod` 在三个状态注入拦截 Action:
    - "Has Dash?" → 检查 Dash Slash (冲斩)
    - "Has Cyclone?" → 检查 Cyclone Slash (旋风斩)
    - "Has G Slash?" → 检查 Great Slash (大招)
  - Action 中发送 "CANCEL" 事件拦截不允许的剑技

### 3.3 具体实现细节

#### NailArtInterceptAction 类
```csharp
public class NailArtInterceptAction : FsmStateAction
{
    public ArrowDirection ExpectedDirection;
    public string NailArtName;
    public Action<NailArtInterceptAction> OnCheck;

    public override void OnEnter()
    {
        if (OnCheck != null)
        {
            OnCheck(this);
        }
    }
}
```

#### 剑技方向映射
| FSM State | Nail Art | 方向 |
|-----------|----------|------|
| "Has Dash?" | Dash Slash | Left/Right (根据 facingRight) |
| "Has Cyclone?" | Cyclone Slash | Up 或 Down |
| "Has G Slash?" | Great Slash | Left/Right (根据 facingRight) |

> **注意**: Dash Slash 和 Great Slash 都使用 Left/Right 方向（基于面朝方向），处理逻辑相同。

#### 拦截逻辑
在 Action 的 OnCheck 回调中：
1. 获取 ArrowGame 组件
2. 调用 `IsSpellAllowed(ExpectedDirection)` 检查
3. 如果不允许，发送 "CANCEL" 事件
4. 如果允许，调用 `OnSuccessfulAction()`
