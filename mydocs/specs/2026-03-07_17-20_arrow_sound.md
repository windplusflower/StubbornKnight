# SDD Spec: 箭头成功/失败音效播放

## 0. 🚨 Open Questions
*None - 需求清晰*

## 1. Requirements (需求)
- **Goal**: 在箭头游戏成功/失败时播放对应音效
- **In-Scope**: 
  - 加载 assets/success.mp3 和 assets/fail.mp3
  - 成功时播放 success.mp3
  - 失败时播放 fail.mp3
- **Out-of-Scope**: 其他音效功能

## 1.5 Code Map (代码拓扑)
- **核心逻辑**:
  - `ArrowGame.cs`: 箭头游戏主逻辑
    - `OnSuccessfulAction()` (第65行): 成功回调 - 添加音效播放
    - `TriggerErrorEffect()` (第89行): 失败回调 - 添加音效播放
  - `StubbornKnight.csproj`: 资源已通过 `<EmbeddedResource Include="assets\**\*" />` 包含
